## 概要
slackのハドルのようなビデオ通話ができるアプリを作成しました。

## 技術スタック

| ---------------- | ------ | ----------------------- |
| **Next.js**      | 15.5.4 | Reactベースのフルスタックフレームワーク  |
| **React**        | 19.1.1 | UI構築ライブラリ               |
| **TypeScript**   | 5.9.2  | 型安全なJavaScript開発        |
| **Tailwind CSS** | 3.4.18 | ユーティリティファーストなCSSフレームワーク |
